## 👥 For Users

### Getting Started

1. **Open the app** (your deployed URL)
2. **Get an API key** from [console.anthropic.com](https://console.anthropic.com/account/keys)
3. **Paste it in Settings** (⚙️ icon)
4. **Generate a practice**
5. **Start your session**

### Usage

```
1. Select desired duration (15-90 min)
2. Choose body focus area
3. Pick background music mood (optional)
4. Click "Generate Practice"
5. Review your personalized sequence
6. Click "Begin Practice" for guided timer mode
```

## 🎯 How It Works

1. User selects preferences (duration, focus, music)
2. App sends request to backend with user's API key
3. Backend proxies to Anthropic Claude
4. Claude generates a custom yoga sequence (JSON format)
5. App displays poses with SVG illustrations
6. User can start guided timer mode
7. Timer auto-advances with alignment cues
8. Session completes with summary
