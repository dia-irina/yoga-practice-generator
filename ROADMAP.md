# Yoga Practice Generator - Feature Ideas & Improvements

## Priority: HIGH (Should add soon)

### User Experience
- [ ] **Save Favorite Practices** — Let users save sequences to localStorage for quick replay
- [ ] **Practice History** — Quick stats: how many practices, total minutes, most-used focus areas
- [ ] **Default Preferences** — Remember user's last selected duration + focus area
- [ ] **Clearer Feedback** — Show "Generating..." spinner with fun yoga affirmations
- [ ] **Error Recovery** — Better recovery from API failures without clicking "Try Again"

### Features
- [ ] **Modify Generated Practice** — Allow editing pose duration or swapping poses before starting
- [ ] **Breathing Guidance** — Visual/audio cues for inhales/exhales during timer
- [ ] **Difficulty Levels** — Easy/Moderate/Advanced pose selection
- [ ] **Pace Control** — Slow down timer if user needs more time in a pose
- [ ] **Pose Notes** — Collapsible tips/modifications for each pose

### Technical
- [ ] **Offline Support** — Cache last practice for offline replay
- [ ] **Dark Mode** — Toggle in settings
- [ ] **Accessibility** — Better keyboard navigation, screen reader support
- [ ] **Performance** — Optimize SVG rendering for smoother animations
- [ ] **Analytics** — Track popular focus areas (fully private, no user tracking)

---

## Priority: MEDIUM (Nice to have)

### Content
- [ ] **More Poses** — Add 10-15 additional poses for variety
- [ ] **Pose Categories** — Group by level (Beginner/Intermediate/Advanced)
- [ ] **Chakra Focus** — Alternative focus on energy centers
- [ ] **Time-of-Day Sequences** — Morning energizers vs evening wind-downs
- [ ] **Props Support** — Suggest modifications with yoga props

### Social Features
- [ ] **Share Practices** — Generate shareable links to same sequence
- [ ] **Practice Comments** — Add notes to saved practices
- [ ] **Community Sequences** — Share top-rated practices (opt-in)

### Integrations
- [ ] **Wearable Integration** — Connect to Apple Watch/Fitbit for heart rate
- [ ] **Calendar Sync** — Add practice to calendar
- [ ] **Smart Home** — Control lights/temperature during practice
- [ ] **Built-in Music** — Play lo-fi or ambient tracks without leaving app
